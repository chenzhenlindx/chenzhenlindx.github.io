发表于 2018-01-22 | 分类于 重构

动态显示

%n$ms：代表输出的是字符串，n代表是第几个参数，设置m的值可以在输出之前放置空格
%n$md：代表输出的是整数，n代表是第几个参数，设置m的值可以在输出之前放置空格
%n$mf：代表输出的是浮点数，n代表是第几个参数，设置m的值可以控制小数位数，如m=2.2时，输出格式为00.00

%1$s

结果：离下班回家还剩38分钟

%1$3s

结果：离下班回家还剩 38分钟
注：m设置为4只有2个空格

%1$2.2f

结果：离下班回家还剩1234.12分钟

使用%%来显示一个%

结果：离下班回家还剩34.12%分钟
